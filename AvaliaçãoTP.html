- O que é Git, Github e modularização?

Git é um controle de versionamento para sistema.

Github é um site que faz o armazenamento dos versionamentos em nuvem

Modularização é "quebrar" um projeto em várias partes, para um melhor controle, correção, e manutenção

- Versão do sistema(1.0, 2.0) é o mesmo que versionar o código?

Versionar é quando os projetos de software são muito grandes e necessitam ser entregues periodicamente.

Essas entregas consistem em partes do sistema pronto e que são criadas as versões e coforme melhoria e ou termino dos demais passos a versão vai sendo alterada e atulizada.

E versão do sistema, é quando ele já foi versionado, testado, e foi finalizado para lançamento.
